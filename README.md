# Social-Network-Analysis-Project
Modeled 24 years of global disaster data across 188 countries, identifying a 29.6% increase in interconnectivity and validating regional climate clusters.  Applied Betweenness Centrality to identify high-risk logistical hubs critical for aid coordination.
Raw data is available here :
https://doc.emdat.be/docs/introduction/
The initial cleaning was done using MS Excel and most data processing was done using R studio, Gephi was used for visualization.
